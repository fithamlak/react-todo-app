# 📖 [react-todo-app] <a name="about-project"></a>

> "react-todo-app" is an application in which user can add,  edit and delete task. taskLists can be stored and saved in localstorage

### Key Features <a name="key-features"></a>

- Set up project using create-react-app 
- Using Components to build the website
- Using localstorage to store todos data on browser
- Using react icons for the website
